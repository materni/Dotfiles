Htop                # Process viewer

Picom              # Compositor

Polybar            # Status Bar

Ranger            # File manager

Zathura           # PDF Reader

Firfox              # Browser

Doas               # Replacement for sudo

Joplin              # Note application

Flameshot      # Screenshot tool

Xorg-xinit        # Launch X11 display server
